Fitness Stopwatch and Rep Counter

I developed a React Native app using React Native Elements. The main focus was creating a Home component with a list of exercise buttons, each leading to a different exercise screen. The DurationExercise, RepetitionExercise, and RepetitionDurationExercise components allow users to start timers, track repetitions, and reset activities.

Basic ToDo List

I developed a simple ToDo app using React Native. I used the FlatList component for task rendering and SafeAreaView for device layout compatibility. Default tasks each included a unique identifier, completion status, and description. They were displayed with a state array. Completed tasks are styled with a strikethrough. Users can type new tasks in an input box. Selecting "Add" places the task in the list.

A challenge I faced was getting the colored check mark boxes to appear when selected.

For the sake of simplicity, I used a monochromatic design to keep the user focused and on task.

Exercise App - Beginning Stage

I created a simple exercise app with multiple screens. The home screen displays categorized exercises as buttons. Clicking a button brings you to the exercise screen with the exercise name displayed at the top. The RepetitionExercise component allows users to increment a repetition counter and reset it, while the DurationExercise component allows users to start and reset a stopwatch. The RepetitionDurationExercise component combines both features.

I struggled getting the stopwatch feature to work properly. I had to use a variety of code combinations to get it to function how I wanted.

This was the beginning concept for a later project. I chose to spend verty little time on the design of the app. I wanted to make sure it was functional first.
