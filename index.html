Fitness Stopwatch and Rep Counter

I developed a React Native app using React Native Elements. The main focus was creating a Home component with a list of exercise buttons, each leading to a different exercise screen. The DurationExercise, RepetitionExercise, and RepetitionDurationExercise components allow users to start timers, track repetitions, and reset activities.

I struggled with the "Suggested" button functionality. I fixed it simplifying the code, as I had made it overcomplicated.

I wanted bright exercise buttons on the home page, but decided to keep a simpler design on individual pages to make it clear to the user how to use the timer and increment features.

I developed a ToDo app using React Native, featuring Login and Registration screens. The Login screen includes fields for username and password, with options for login and registration. I chose to keep the colors consitent across all pages. I used brighter colors to help the buttons stand out.

Basic ToDo List

I developed a simple ToDo app using React Native. I used the FlatList component for task rendering and SafeAreaView for device layout compatibility. Default tasks each included a unique identifier, completion status, and description. They were displayed with a state array. Completed tasks are styled with a strikethrough effect, and users can add new tasks in an input box.

A challenge I faced was getting the colored check mark boxes appeared when selected.

For the sake of simplicity, I used a monochromatic design to keep the user focused and on task.

Exercise App - Beginning Stage

I created a simple exercise app with multiple screens. The home screen displays categorized exercises as styled buttons. Clicking a button brings you to the exercise screem with the exercise name shown at the top. The RepetitionExercise component allows users to increment a repetition counter and reset it, while the DurationExercise component allows users to start and reset a stopwatch. The RepetitionDurationExercise component combines both features.

I struggled getting the timer feature to work properly. I had to use a variety of code combinations to get the timer to function how I wanted.

This was the beginning concept for a later project. I chose to spend verty little time on the design on the app. I wanted to make sure it was functional first.
